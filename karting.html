Karting Game
This is a 3D, low-poly racing game made with Unity and Blender. You play as a kart and you race around the track trying to set a best time. The game keeps track of how many laps you have completed, as well as your all time best lap time and your current lap time. Check out more of the project on GitHub or try out the game here.

Usage

This repository contains everything needed to get the project up and running on Unity. Just clone the repository and open the project in Unity to edit the game yourself. The kart can be driven around with basic arrow keys or WASD keys.

Limitations

Multiplayer - The game cannot be played online.
ML Bots - The game does not feature bots to race against.
Platform - The game can only be played on a computer.
