Kloto

Kloto is a single-player, FPS portal simulator. The game consists of two-way and one-way portals to teleport through. To render the portal screens, we used Unity's Universal Render Pipeline, render textures, shaders, and linear algebra. The gameplay is enhanced with immersive visual special effects, sound effects, and two unique maps. You can check more of the project out on GitHub or give the game a try here.

Usage

The player can be moved around using WASD or the arrow keys. To jump, press the space bar. Shooting can be done through the left and right mouse click.


Limitations

Platform - The game can only be played on a computer
